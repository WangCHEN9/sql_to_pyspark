# sql_to_pyspark

sql to pyspark converter , build with [streamlit](https://streamlit.io/)

Original from [sqlandhadoop](https://sqlandhadoop.com/online-sql-to-pyspark-converter/)

## Covered / Uncoverd SQL commands

### Covered

- FROM
- WHERE
- GROUPBY
- AGG
- SELECT
- SORT
- LIMIT

### Uncovered

- JOIN
- SUBQUERIES
